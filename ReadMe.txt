Shopee Web Developer Test

How to set up:
1. Download all the files in GitHub or download Shopee.zip
2. Copy and paste all the files into a folder name Shopee in your localhost
3. Download and import shopee.sql to your MySQL admnistration tool
4. Open the website in your browser by inputting the url localhost/Shopee/ or localhost/Shopee/index.p

The programming languages used in this project: HTML, CSS, Javascript, PHP, SQL 
Front end framework: Bootstrap
